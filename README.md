# Doodle Jump (Python + Pygame)

## Overview

Doodle Jump is an arcade-style platform jumping game developed in Python using the Pygame library. The player continuously jumps between platforms while the game scrolls upward, earning points by surviving and reaching higher positions.

## Features

* Platform Jumping Mechanics
* Gravity-Based Movement
* Infinite Vertical Scrolling
* Random Platform Placement
* Collision Detection
* Score Tracking
* Screen Wrapping
* Game Over Screen
* Restart Functionality

## Technologies Used

* Python 3.12
* Pygame

## Controls

* Left Arrow → Move Left
* Right Arrow → Move Right
* Enter → Restart Game

## Installation

### Install Pygame

```bash
pip install pygame
```

### Run the Game

```bash
python doodle_jump.py
```
